Introduction à JavaScript

JavaScript est un langage de programmation principalement utilisé pour rendre les pages web interactives.

Variables et Types de Données

En JavaScript, on utilise let, const, et var pour déclarer des variables.

let nom = "Alice"; // chaîne de caractères
const age = 30;    // nombre
let estEtudiant = true; // booléen

Essayez dans la console pour voir les types de données et manipuler des variables.

Opérateurs

Les opérateurs permettent de manipuler les valeurs en JavaScript.

let x = 5;
let y = 3;
let sum = x + y; // addition : 8
let diff = x - y; // soustraction : 2
let prod = x * y; // multiplication : 15
let quotient = x / y; // division : 1.666...

Structures de Contrôle

Les conditions if permettent de prendre des décisions dans le code.

let heure = 14;
if (heure < 12) {
    console.log("Bonjour !");
} else {
    console.log("Bon après-midi !");
}

Les boucles for permettent de répéter une action plusieurs fois.

for (let i = 0; i < 5; i++) {
    console.log("Nombre: " + i);
}

Fonctions

Les fonctions permettent de réutiliser du code facilement.

function saluer(nom) {
    return "Bonjour, " + nom + "!";
}
console.log(saluer("Alice")); // Affiche "Bonjour, Alice!"

Manipulation du DOM

JavaScript peut interagir avec le contenu HTML d'une page grâce au Document Object Model (DOM).

            // Manipulation du DOM : Changer le titre
function changeTitle() {
    document.querySelector("header h1").innerText = "Cours de JavaScript Modifié!";
}

Événements

Les événements permettent d'interagir avec l'utilisateur, par exemple avec des clics ou des touches du clavier.

document.getElementById("monBouton").addEventListener("click", function() {
    alert("Bouton cliqué !");
});

Async / Await et Promises

Les Promises permettent de gérer des opérations asynchrones, et async/await simplifie leur utilisation.

async function fetchData() {
    let response = await fetch("https://jsonplaceholder.typicode.com/todos/1");
    let data = await response.json();
    console.log(data);
}
fetchData();
