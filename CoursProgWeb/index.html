Media Queries

Les media queries permettent d’adapter le style CSS en fonction de la taille de l’écran. Par exemple, elles permettent de changer la mise en page sur les appareils mobiles.

@media (max-width: 768px) {
    body {
        background-color: #e0e0e0;
    }
    h1 {
        font-size: 1.5em;
    }
}

Dans cet exemple, les styles seront appliqués uniquement si la largeur de l’écran est inférieure ou égale à 768 pixels.

Flexbox

Flexbox est un modèle de mise en page qui permet d’organiser des éléments de manière flexible. Il est utile pour créer des mises en page réactives et aligner des éléments.

.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

Dans cet exemple, les éléments à l'intérieur de la .container sont centrés horizontalement et verticalement.
